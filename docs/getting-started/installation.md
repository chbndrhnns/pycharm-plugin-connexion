# Installation

There are several ways to install the Python BetterPy plugin.

## IDE Built-in Plugin System (Recommended)

The easiest way to install is through the IDE's built-in plugin system. This ensures you receive automatic updates.

1. Open **Settings/Preferences** (++ctrl+alt+s++ or ++cmd+,++).
2. Navigate to **Plugins**.
3. Select the **Marketplace** tab.
4. Search for `Python BetterPy`.
5. Click **Install**.
6. Restart the IDE if prompted.

!!! tip
    This is the recommended way as it handles dependencies and updates automatically.

## Manual Installation

If you need to install a specific version or don't have internet access from your IDE:

1. Download the plugin ZIP file from the [GitHub Releases](https://github.com/chbndrhnns/pycharm-ddd-toolkit/releases) or the [JetBrains Marketplace](https://plugins.jetbrains.com/plugin/23246-python-ddd-toolkit).
2. Open **Settings/Preferences** > **Plugins**.
3. Click the gear icon (⚙️) and select **Install Plugin from Disk...**.
4. Select the downloaded ZIP file.
5. Restart the IDE.

## Compatibility

- **Required IDE**: PyCharm 2023.1+ or IntelliJ IDEA 2023.1+.
- **Required Plugins**: In IntelliJ IDEA, the **Python** plugin must be installed.
