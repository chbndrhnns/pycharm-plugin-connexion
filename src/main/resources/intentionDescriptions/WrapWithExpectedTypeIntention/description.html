Wraps an expression causing a type mismatch with the expected type constructor.

When a type mismatch is detected (e.g., expected str but got Path), this intention automatically wraps the problematic expression with the expected type constructor.

Example:

Before After 
a: str = Path("val")
 
a: str = str(Path("val"))

This helps quickly fix type annotation mismatches by automatically applying the appropriate type conversion.
