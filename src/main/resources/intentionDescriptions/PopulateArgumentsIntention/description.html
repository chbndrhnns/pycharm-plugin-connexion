Populates missing call arguments with placeholder ellipsis values (...).

Shows a popup chooser allowing you to select:

  • All arguments - populates all missing arguments including those with default values
  • Required arguments only - populates only required arguments (without defaults)
  • Recursive - expands nested dataclasses/Pydantic models
  • Use constructors - uses int()/str() placeholders instead of ... for builtins

This intention is available when the caret is inside a call expression with missing arguments.
