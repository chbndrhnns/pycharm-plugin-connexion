Unwraps a non-container wrapper call (for example, a NewType or value object constructor) when the inner expression already matches the expected type at the current position.

Examples:

  • x: int = UserId(42) → x: int = 42
  • f(user_id: int); f(UserId(10)) → f(10)
  • def get_user_id() -> int: return UserId(1) → def get_user_id() -> int: return 1
