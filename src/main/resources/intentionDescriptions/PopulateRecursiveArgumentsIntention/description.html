Populate missing arguments in a function or class call, recursively instantiating nested submodels (for example dataclasses or Pydantic models) with placeholder values (...).

Example:

@dataclass
class Address:
    street: str
    city: str

@dataclass
class User:
    id: int
    name: str
    address: Address

user = User(...)  # caret is inside the argument list
becomes: 
user = User(
    id=...,
    name=...,
    address=Address(
        street=...,
        city=...,
    ),
)

This helps quickly scaffold complex object creation code while making all required fields explicit. You can then replace the ellipses with real values.
