Populate missing call arguments (including keyword-only parameters) at the caret with ellipsis values (...).

For example, given a dataclass constructor or function call with the caret in the argument list, the intention will add keyword arguments for all unmapped parameters: 

@dataclass
class A:
    x: int
    y: int

a = A(<caret>)
becomes: 
a = A(x=..., y=...)
