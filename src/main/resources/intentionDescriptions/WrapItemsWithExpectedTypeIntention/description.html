Wraps each element in a container expression with the expected item type constructor.

When a type mismatch is detected for a container (for example, the expected type is list[UserId] but the actual literal is [1, 2, 3]), this intention offers to wrap each element with the expected constructor.

Example:

Before After 
from mytypes import UserId

users: list[UserId] = [1, 2, 3]
 
from mytypes import UserId

users: list[UserId] = [UserId(1), UserId(2), UserId(3)]

This helps quickly fix container type mismatches by automatically applying the appropriate constructor to each item.
