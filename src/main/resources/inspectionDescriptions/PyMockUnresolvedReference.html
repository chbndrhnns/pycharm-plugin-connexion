Reports unresolved attribute references on Mock objects that have a spec.

When a Mock object is configured with a spec, it is restricted to the attributes and methods present in that spec. This inspection flags any attempt to access an attribute or method that does not exist in the specified class or object.

Example:


class MyClass:
    def existing_method(self):
        pass

mock = Mock(spec=MyClass)
mock.existing_method()  # OK
mock.non_existent()      # Unresolved attribute reference 'non_existent' for class 'MyClass'
