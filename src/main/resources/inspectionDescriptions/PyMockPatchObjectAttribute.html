Reports unresolved attribute references in unittest.mock.patch.object(target, "attribute") calls.

When using patch.object, the attribute name (second argument) should exist on the target class. This inspection verifies that the specified attribute is actually defined on the target.

Example:


class MyClass:
    def existing_method(self): pass

# OK - 'existing_method' exists on MyClass
patch.object(MyClass, "existing_method")

# Warning - 'non_existing' does not exist on MyClass
patch.object(MyClass, "non_existing")
