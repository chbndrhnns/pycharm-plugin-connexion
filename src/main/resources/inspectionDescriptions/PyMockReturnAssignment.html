Reports invalid assignments to return_value on Mock objects when the assigned type does not match the expected return type of the mocked callable.

When a Mock object is configured with a spec that is a callable (like a function or a class), this inspection checks if the value assigned to its return_value attribute matches the return type defined in the spec.

Example:


def get_name() -> str:
    ...

mock = Mock(spec=get_name)
mock.return_value = 123  # Expected type 'str', got 'int' instead
